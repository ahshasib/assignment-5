Question-1: What are the different ways to select an element in the DOM?

Answer: There have several ways to select elements in the DOM using JavaScript.Like: getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll, getElementsByName.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:Using innerHTML you can returns or sets HTML content inside an element.Using innerText you can returns or sets only visible text inside an element.Using textContent you can returns or sets all text content inside an element.

Question-3: What is event delegation in the DOM?

Answer:Event Delegation is a technique in JavaScript where instead of attaching event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Answer:Event Bubbling is a behavior in the DOM where an event triggered on a child element bubbles up (propagates) to its parent and then to higher ancestors, unless explicitly stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer:Use document.createElement() to create a new HTML element.There have maney ways to add the new element to the DOM:appendChild(), append(), prepend(), insertBefore(). For remove elements you can use removeChild(), remove()
